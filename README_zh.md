# node-search-file

遍历文件夹中的文本文件，快速定位关键词

## 安装

``` npm install -g node-search-file ```

## 使用

打开命令行终端，切换到你想要寻找内容的目录

输入命令：

``` search ${查找的目录名称}  ${关键词} ${是否忽略大小写} ```

终端将输出类似下面格式的结果：

```
	[dir]: ./lib
	[---]: ./lib/index.js
```
